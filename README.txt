------------------Program Description-----------------
1. calcpi_serial.C
	Estimating value of pi using the midpoint rule

2. calcpi_omp.C
	Parallel implementation for estimating value of pi using the midpoint rule using OpenMP

3. calcpi_mpi.C
	Parallel implementation for estimating value of pi using the midpoint rule using MPI
------------------------------------------------------

-----------------Running the code---------------------
No command line arguments are required. timer.C needs to be compiled along with the source code for the serial and OpenMP versions.
------------------------------------------------------
